# XPLORER

XPLORER is a smart tourist assistant. It goes beyond simple recommendations by creating time‑optimised day plans that tell you when and in what order to visit places minimising travel time and avoiding crowds. It also handles hotel and transport bookings seamlessly.

# Features
- Intent‑aware conversation – Understands whether you want to plan a trip, book a hotel, or both.
- Personalised recommendations – Uses vector search (semantic similarity) to match your interests.
- Optimal daily flow – Graph‑based sequencing ensures you visit attractions in the right order at the best times.
- One‑click bookings
